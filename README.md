---

## 📌 What is Pandas?

**Pandas** is a fast, powerful, and flexible Python library for data analysis and manipulation.  
It provides two key data structures:

- `Series`: 1D labeled array  
- `DataFrame`: 2D labeled table (like a spreadsheet)
